# Spoton - Spotify now-playing for mIRC  
A program written in C++ that make Spotify now-playing for mIRC. 
  
### This version of Spoton supports
  
mIRC Version: 7.64 - 7.67  
Spotify Version: 1.1.51 - 1.1.69
